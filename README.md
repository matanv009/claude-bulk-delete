# Claude Bulk Delete Chats

A Chrome extension that adds a **bulk delete** button to [Claude.ai](https://claude.ai), letting you select and delete multiple chats at once.

## Why?

Claude doesn't have a built-in way to delete multiple chats. This extension adds a simple UI to do it in seconds.

## Installation

> The extension is not on the Chrome Web Store — install it manually in 3 steps.

1. **Download** — Click the green **Code** button above → **Download ZIP** → Extract the folder
2. **Open Chrome extensions** — Go to `chrome://extensions` in your browser
3. **Load the extension** — Enable **Developer mode** (top right) → Click **Load unpacked** → Select the extracted folder

That's it. Refresh [claude.ai](https://claude.ai) and you'll see the **🗑 Bulk Delete Chats** button in the sidebar.

## How to use

1. Click **🗑 Bulk Delete Chats** in the sidebar
2. Check the chats you want to delete (or click **Select All**)
3. Click **Delete (N)** and confirm

## Notes

- Works on `claude.ai` only
- Tested on Chrome and Edge
- No data is collected — everything runs locally in your browser
